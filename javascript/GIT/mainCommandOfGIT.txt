Полезные команды консоли:
cd - переход в заданную папку
ls - показывает список файлов текущей папки
mkdir - создать папку
rmdir - удаление пустой папки
touch - создать файл
mv - перемещение файлов/папок
rm - удаление файлов/папок

Команды git:
git init - инициализирует локальный репозиторий в папке
git clone - клонирует удаленный репозиторий в текущую папку. Например - git clone "ssh" "name of new folder"
git status - отображает состояние рабочего каталога и stage
git add "file name"; git add "." - add all files to stage - добавляет новые или измененные файлы в stage
git reset - удаляет файл из под git-контроля. Пример - git reset HEAD "app.js"
git commit - делает коммит подготовленных файлов в историю проекта
git push - отправляет историю проекта (коммиты) на удаленный репозиторий
git log - отобржает историю коммитов локального репозитория
git remote -v - отображает с каким удаленным репозиторием связан локальный
git branch - отображает список существующих веток
git branch -d "branch name" - удаляет указанную ветку. Если указать -D, тогда ветка несмержиная ветка будет удалена без дополнительного подтверждения
git checkout -b "new branch name" - создание новой ветки
git checkout "branch name" - для перехода между ветками
git merge "branch name" - для слияния веток, переходим в ветку, в которую хотим слить информацию и вводим указанную команду
git push -u origin "branch name" - заливает новую ветку с локального репозитория на удаленный и начинает отслеживать
git pull - забирает с удаленного репозитория последние обновления


Другие команды, информация:

code "." - открывает редактор кода
clear - очищает рабочую область в терминале
хорошая практика называть репозитории с маленькой буквы, через "-"
# - текстовая информация (комментарий)
$ - обозначение начала консоли (в терминале)
> - output (обозначение результата работы)
